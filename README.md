# 90 Days of AI Blog Series

This folder contains the working structure for a 90-day daily AI blog series.

## Structure

- `topics/topics.json` - full 90-day topic calendar
- `blogs/day-XX/` - one folder per daily blog
- `blogs/day-XX/blog.md` - publishable Markdown article
- `blogs/day-XX/metadata.json` - title, SEO, keywords, and publishing notes
- `blogs/day-XX/assets/` - diagrams, hero images, and graphics
- `blogs/day-XX/examples/` - runnable code samples used in the article

## Style Guide

- Explain AI topics like the reader is smart but new.
- Use practical examples before abstract theory.
- Add light comic flavor, but keep the technical explanation focused.
- Prefer diagrams for flows and mental models.
- Include code only when it makes the concept clearer.
- Make every article useful on its own, but connected to the full 90-day learning path.
